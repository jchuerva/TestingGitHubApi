# TestingGitHubApi
Tool for extract information from GitHub API

### Installation

Bundler gem needs to be previously installed. More info [here](https://github.com/bundler/bundler)

Install all gems in the root folder


```sh
  bundle install
```

### How to use it


```sh
  ruby app.rb
```
